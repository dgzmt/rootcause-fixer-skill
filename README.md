# Rootcause Fixer Skill

A structured problem-solving skill for deep root cause analysis and comprehensive solution development.

## Overview

Rootcause Fixer is a skill designed to help you analyze problems systematically, identify their underlying root causes, and develop effective, long-lasting solutions. It uses a proven 5-step methodology to guide you from problem identification to actionable solutions.

## Features

- **5-Step Analysis Process**: Systematic approach to problem solving
- **5W1H Definition**: Clear problem boundary identification
- **Fishbone Diagram**: Multi-dimensional cause exploration
- **5Why Deep Dive**: Root cause penetration
- **PDCA Solutions**: Both tactical and strategic fixes

## Installation

### For Trae IDE

1. Copy the `.trae/skills/rootcause-fixer/` directory to your project
2. The skill will be automatically detected and available

## Usage

### Basic Usage

Invoke the skill in your conversation:

```
@rootcause-fixer [your problem description]
```

### Example

```
@rootcause-fixer My website is loading very slowly and affecting user experience
```

## Methodology

### Step 1: Problem Definition (5W1H)
- **What**: Core phenomenon
- **Where/When**: Spatial and temporal boundaries
- **Who**: Stakeholders
- **How Much**: Quantified impact

### Step 2: Multi-dimensional Exploration (Fishbone)
- Technical categories: Man, Machine, Material, Method, Environment, Measurement
- Management categories: Strategy, Organization, Process, People, Culture, System
- Personal growth categories: Cognition, Skills, Resources, Mindset, Environment, Action

### Step 3: Deep Root Tracing (5Why)
- Ask "Why" at least 5 times
- Go beyond "human error" to system/process issues

### Step 4: Hypothesis Validation
- Form clear validation hypotheses
- Design minimal verification schemes

### Step 5: Solution Implementation (PDCA)
- Tactical fixes (immediate)
- Strategic solutions (long-term)
- Knowledge asset
